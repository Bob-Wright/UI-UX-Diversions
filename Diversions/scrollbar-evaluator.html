More information about the width of browser scrollbars can be found here. Thanks TextFixer!

This page was adapted from the demo scrollbar-pixel-width-test from that site.

In addition to adding the display of the browser window size, the display of the contents that we query are left visible and
normalizing CSS was used to defeat the default margins and padding so the scrollbar is at the viewport edge
without any offsets/displacement. Also the value (in pixels) of the scrollbar width is rounded to the nearest integer value.
